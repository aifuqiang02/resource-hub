# 臻享资源 (ZhenXiangZiYuan)

优质资源导航与发现平台，解决网上资源太多、难以找到有效资源的问题。

## 项目简介

帮助用户发现和分享优质资源的导航平台。

- **目标用户**：需要寻找学习资源、技术资料的用户
- **核心功能**：资源导航、智能推荐、资源评价、社区分享
- **部署状态**：规划中

## 技术栈

- **前端**：Next.js + React
- **后端**：Node.js + Python
- **搜索**：Elasticsearch
- **AI**：大语言模型（智能推荐）
- **数据库**：PostgreSQL + Redis

## 端口规划（40250-40299）

| 端口 | 服务 | 环境 | 描述 |
|-----|------|------|------|
| 40250 | web | dev | Web主站 |
| 40251 | api | dev | API接口服务 |
| 40252 | search | dev | 搜索服务 |
| 40253 | ai | dev | AI推荐服务 |

## 核心功能

1. **资源导航** - 按类别整理优质资源
2. **智能搜索** - 快速找到所需资源
3. **AI推荐** - 基于用户需求推荐资源
4. **资源评价** - 用户评分和评论
5. **收藏分享** - 收藏和分享优质资源
