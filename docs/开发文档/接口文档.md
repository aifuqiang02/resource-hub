# 接口文档

## 基础信息

- **Base URL**: `/api/v1`
- **认证方式**: Bearer Token
- **Content-Type**: `application/json`

## 通用响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

## 认证接口

### 用户注册

```
POST /auth/register
```

**请求体：**

```json
{
  "username": "string",
  "email": "string",
  "password": "string"
}
```

**响应：**

```json
{
  "code": 201,
  "message": "注册成功",
  "data": {
    "user": {
      "id": "uuid",
      "username": "string",
      "email": "string"
    },
    "token": "string"
  }
}
```

### 用户登录

```
POST /auth/login
```

**请求体：**

```json
{
  "email": "string",
  "password": "string"
}
```

**响应：**

```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "string",
    "user": {
      "id": "uuid",
      "username": "string",
      "email": "string"
    }
  }
}
```

## 资源接口

### 获取资源列表

```
GET /resources
```

**查询参数：**

| 参数 | 类型 | 说明 |
|------|------|------|
| q | string | 关键词搜索 |
| type | string | 资源类型 |
| sort | string | 排序字段 |
| order | string | 排序方式 |
| page | number | 页码 |
| page_size | number | 每页数量 |

**响应：**

```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "id": "uuid",
        "name": "资源名称",
        "description": "资源描述",
        "rating": 4.5,
        "downloads": 1000,
        "created_at": "2024-01-01T00:00:00Z"
      }
    ],
    "total": 100,
    "page": 1,
    "page_size": 20
  }
}
```

### 获取资源详情

```
GET /resources/:id
```

**响应：**

```json
{
  "code": 200,
  "data": {
    "id": "uuid",
    "name": "资源名称",
    "description": "资源描述",
    "version": "1.0.0",
    "rating": 4.5,
    "downloads": 1000,
    "comments_count": 50,
    "created_at": "2024-01-01T00:00:00Z"
  }
}
```

## AI 搜索接口

### AI 智能搜索

```
POST /ai/search
```

**请求头：**

```
Authorization: Bearer <token>
```

**请求体：**

```json
{
  "message": "我需要一个图片处理工具",
  "history": [
    {
      "role": "user",
      "content": "我需要开发工具"
    },
    {
      "role": "assistant", 
      "content": "为您推荐以下开发工具..."
    }
  ]
}
```

**响应：**

```json
{
  "code": 200,
  "data": {
    "message": "为您推荐以下资源：",
    "resources": [
      {
        "id": "uuid",
        "name": "图片处理工具 A",
        "description": "功能强大的图片处理工具",
        "rating": 4.5,
        "downloads": 1000
      },
      {
        "id": "uuid",
        "name": "图片处理工具 B",
        "description": "轻量级图片处理工具",
        "rating": 4.2,
        "downloads": 500
      }
    ]
  }
}
```

## 收藏接口

### 获取收藏列表

```
GET /user/favorites
```

**请求头：**

```
Authorization: Bearer <token>
```

**响应：**

```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "id": "uuid",
        "resource_id": "uuid",
        "resource_name": "资源名称",
        "created_at": "2024-01-01T00:00:00Z"
      }
    ]
  }
}
```

### 收藏资源

```
POST /resources/:id/favorite
```

**请求头：**

```
Authorization: Bearer <token>
```

### 取消收藏

```
DELETE /resources/:id/favorite
```

**请求头：**

```
Authorization: Bearer <token>
```

## 下载接口

### 获取下载链接

```
GET /resources/:id/download
```

**请求头：**

```
Authorization: Bearer <token>
```

**响应：**

```json
{
  "code": 200,
  "data": {
    "url": "https://cdn.example.com/file.zip",
    "expires_at": "2024-01-01T01:00:00Z"
  }
}
```

## 错误码

| 码 | 说明 |
|-----|------|
| 200 | 成功 |
| 201 | 创建成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |
