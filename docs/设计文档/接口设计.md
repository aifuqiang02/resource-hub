# 接口设计

## 1、API 规范

- **基础路径**: `/api/v1`
- **认证方式**: Bearer Token
- **数据格式**: JSON
- **响应格式**: 统一响应包装

## 2、认证接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /auth/register | 用户注册 |
| POST | /auth/login | 用户登录 |
| POST | /auth/logout | 登出 |
| GET | /auth/profile | 获取当前用户信息 |

## 3、资源接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /resources | 获取资源列表（支持搜索、筛选、排序） |
| GET | /resources/:id | 获取资源详情 |
| POST | /resources | 创建资源（管理员） |
| PUT | /resources/:id | 更新资源（管理员） |
| DELETE | /resources/:id | 删除资源（管理员） |

### 资源搜索参数

| 参数 | 类型 | 说明 |
|------|------|------|
| q | string | 关键词搜索 |
| type | string | 资源类型筛选 |
| sort | string | 排序字段 (downloads/rating/created_at) |
| order | string | 排序方式 (asc/desc) |
| page | number | 页码 |
| page_size | number | 每页数量 |

## 4、AI 搜索接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /ai/search | AI 智能搜索（对话形式） |

### AI 搜索请求

```json
{
  "message": "我需要一个图片处理工具",
  "history": [
    {"role": "user", "content": "我需要开发工具"},
    {"role": "assistant", "content": "为您推荐以下开发工具..."}
  ]
}
```

### AI 搜索响应

```json
{
  "message": "为您推荐以下资源：",
  "resources": [
    {
      "id": "uuid",
      "name": "图片处理工具 A",
      "description": "功能强大的图片处理工具",
      "rating": 4.5,
      "downloads": 1000
    }
  ]
}
```

## 5、评价接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /resources/:id/ratings | 评分 |
| POST | /resources/:id/comments | 评论 |
| GET | /resources/:id/comments | 获取评论列表 |

## 6、用户收藏接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /user/favorites | 获取收藏列表 |
| POST | /resources/:id/favorite | 收藏 |
| DELETE | /resources/:id/favorite | 取消收藏 |

## 7、下载接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /resources/:id/download | 获取下载链接（需登录） |

## 8、错误响应格式

```json
{
  "code": 400,
  "message": "错误信息",
  "details": {}
}
```

### 错误码

| 码 | 说明 |
|-----|------|
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |
