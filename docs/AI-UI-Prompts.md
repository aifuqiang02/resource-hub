# AI UI Prompts

本文件包含用于 AI 生成页面的 Prompt，可直接用于 Stitch `generate_screen_from_text`。

## UI 主题

现代简洁的软件资源下载平台风格。

### 色彩规范

- 主色: #2563EB（蓝色，主要按钮、链接）
- 辅色: #64748B（次要文字、图标）
- 背景色: #F8FAFC（页面背景）

### 组件规范

- 主按钮: 主色填充，白色文字，圆角 6px
- 次按钮: 透明背景，主色边框，圆角 6px
- 卡片: 白色背景，圆角 8px，轻微阴影

## 页面清单

```
一级页面
├── 首页：导航栏、搜索框、AI助手入口、搜索结果、Footer
└── 资源详情：资源信息、评分评论、下载按钮、收藏分享

二级页面
├── AI助手对话框：悬浮按钮、对话界面、推荐资源卡片
├── 登录页：登录表单
├── 注册页：注册表单
├── 个人中心：用户信息、积分余额、收藏列表、下载历史
├── 积分充值页：余额显示、充值金额选项、支付二维码
├── 资源上传页：上传表单、文件选择、审核说明
└── 联系我们页：微信群二维码、加入说明
```

## 一级页面

### 页面: 首页

**项目背景**: 臻享资源是一个优质软件资源导航与发现平台，用户可通过关键词搜索或AI助手智能推荐找到所需软件。下载需消耗积分，上传资源可获得积分。

资源下载平台首页，包含：顶部导航栏、中央搜索框、AI助手入口、搜索结果列表、Footer含微信群二维码

### 页面: 资源详情

资源详情页面，包含：资源信息展示区、评分评论区块、下载按钮、收藏按钮、分享按钮

## 二级页面

### 页面: AI助手对话框

AI助手对话弹窗，包含：对话消息区、推荐资源卡片列表、用户输入框、发送按钮

### 页面: 登录页

用户登录页面，包含：用户名输入框、密码输入框、登录按钮、注册链接

### 页面: 注册页

用户注册页面，包含：用户名输入框、邮箱输入框、密码输入框、确认密码输入框、注册按钮、登录链接

### 页面: 个人中心

用户个人中心页面，包含：用户头像和昵称、积分余额展示、我的收藏列表、下载历史记录

### 页面: 积分充值页

积分充值页面，包含：当前积分余额显示、充值金额选项按钮、支付二维码图片、充值说明文字

### 页面: 资源上传页

资源上传页面，包含：资源名称输入框、资源描述文本框、文件上传区域、上传按钮、审核状态说明

### 页面: 联系我们页

联系我们页面，包含：微信群二维码图片、加群说明文字、联系邮箱
